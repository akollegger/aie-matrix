# @aie-matrix/random-agent

Wanderer-tier reference agent. Serves an A2A card at `/.well-known/agent-card.json` and a JSON-RPC A2A endpoint; runs a random `whereami` / `exits` / `go` loop on spawn. See `specs/009-ghost-house-a2a/quickstart.md` for register + spawn + TCK.

## Develop

```bash
pnpm --filter @aie-matrix/random-agent dev
```
